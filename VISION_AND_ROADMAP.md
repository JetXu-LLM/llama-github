# Vision and Roadmap

This document is kept as historical project context.

`llama-github` is now maintained as a focused foundation library rather than an actively expanding roadmap-driven product. If you are evaluating the repository today, prefer the current README and documentation over the historical ambition captured in earlier versions of this file.

Current maintenance priorities are:

- keep the public API stable
- keep packaging and tests healthy
- keep retrieval and PR-content workflows reliable
- keep examples and docs aligned with actual behavior
